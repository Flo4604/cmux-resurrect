## What Changed in v1.6.0

### Tab Completion Engine (NEW — hero feature)
The interactive shell now has full tab completion with 3-level depth:
- Press **Tab** to see available commands with icons and descriptions
- **Tab/Shift-Tab/Up/Down** cycle through options
- **Escape** navigates back to the parent level
- Works at all levels: commands → subcommands → arguments
- Layout names, template names, and Blueprint entries auto-complete

### Settings Hierarchy (NEW)
- New `settings` command group in the TUI
- Banner style configuration moved under `settings > banner > set/get/list`
- Group commands (`bp`, `settings`) show subcommands when you press Enter

### UX Polish
- **Phoenix bye messages**: 15 random farewell messages on exit, printed to the terminal
- **Command header**: dim line showing what command was just executed
- **Smart prompt retention**: usage errors keep the command so you can fix args without retyping
- **Browse mode**: prompt always visible at top during list navigation
- **Confirmation dialogs**: destructive operations (delete, bp remove) ask before executing
